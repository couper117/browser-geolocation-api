 # Global Distance Calculator

## Project Description
GeoGreen is a professional dashboard that calculates the geographical distance between your current location and any city in the world. It utilizes the **Browser Geolocation API** for tracking and **OpenStreetMap** for geocoding place names.

## Features
*Live GPS Tracking: Finds your current location automatically.
*Search Anywhere: Type any city name to calculate distance.
*Interactive Map: Visualizes the route using Leaflet.js.

## How to Use
1. Open the App
2. Search
3. Calculate
4. Allow Access
5. View Results
